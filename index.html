Introduction

These buttons can be clicked to play back a chord where each value in the vector cooresponds to particular note in a 12-tone equal temperament system. These vectors are a representation of chords as vectors under a linearization. The linearization is done by setting A4 = 440 and using a formula to convert frequencies into indices that denote the number of semitones the pitch is away from A4.


i(h) = 12log2(h/440)


h(i) = 440(2)i/12


Here, i denotes the frequency index, which is the difference in semitones between the pitch and A4. h denotes the frequency of the note. When the vector is played back, the indices map bijectively to a frequency, and those frequencies are played back. Please note that the A4 = 440 value may be adjusted as needed for different tuning standards.

Vector Space Axioms

The conversion from absolute frequencies to indices is necessary for the chords to satisfy properties of vector spaces.

For example, chords defined as a vector of their frequencies would not be closed under scalar multiplication, since the zero vector is not included. A chord must contain at least two notes, and having a note with zero frequency is equivalent to not having a note. Converting frequencies to indices resolves much of these issues by naturally allowing for any value to be an index, including non-integer values, which can represent microtones or different tuning systems.

Naturally, the 0 vector exists within this vector space, as the result of all voices union on the same A4.




The additive inverse of a chord can simply be found by negating all entries. Adding a chord and its additive inverse together will produce the 0 vector. For example, the E major chord from the introduction and its additive inverse are shown below.




The additive inverse of the E major chord is a G minor chord. In this case, it is first inversion. This also gives a mathematical representation of mirror harmony where A4 is the mirror point.


The remaining axioms are relatively well defined already, since the properties of scalar multiplication with distribution remains closed, as well as associativity, commutativity, and the identity element.

Experiments

As mentioned before, the negative chord with respect to A4 can be found by scaling the chord vector by -1. Let's try this with some other chords.







A notable detail that can be observed from taking the additive inverse of various chords is that major chords and minor chords are additive inverses of each other. The inverses of augmented and diminished chords, however, are augmented and diminished chords respectively. Let's try to prove these observations with some amount of mathematical rigor.


Let's try and show that the additive inverse of a major chord is a minor chord. For simplicity, we will only evaluate this for a closed-voiced major chord in root position. Start by defining it parametrically with respect to the root r1. This results in the vector [r1, r1 + 4, r1 + 7, r1 + 12]. Note that r1 can be any real number.


A minor chord can be represented with respect to a root r2 as v2 = [r2, r2 + 3, r2 + 7, r2 + 12]. Taking the additive inverse of the parametrically-defined major chord from the previous step yields v1 = [-r1, -r1 - 4, -r1 - 7, -r1 - 12]. Note that taking the negative of a chord vector also flips the position of all voices, so the lowest voice becomes the highest voice, the highest voice becomes the lowest voice, and so on.


Four properties are key to this proof. First, we can freely add any integer multiple of 12 to any entry in the vector while maintaining major or minor tonality. This follows from how inverted chords are constructed. Secondly, adding or subtracting the same real quantity from all entries shifts the chord by a certain number of semitones, which also preserves major and minor tonality. Thirdly, the order of the entries does not matter for defining tonality. As long as the set of indices is the same, it has the same tonality. Finally, if two entries differ by a multiple of 12, they coorespond to the same note at different octaves, which means that one of the notes has no bearing on the tonality of the chord.


Using these facts, add 7 to all entries in v1 to get [-r1 + 7, -r1 + 3, -r1, -r1 - 5], which maintains the tonality of the original chord. Notice that the first and last entries differ by 12, meaning that they coorespond to the same note at different octaves. Set r1 = -r2, yielding the vector [r2 + 7, r2 + 3, r2, r2 - 5]. Looking at the entries, we can see that the first three entries in v2 match those in v1, albeit in a different order. The remaining entry entry r2 - 5 represents the note r2 + 7 at a different octave. In v2, r2 + 12 is also just r2 at a different octave. Neither of those notes have any bearing on the tonality of the chord, so they can be ignored. Therefore, the additive inverse of a major chord is a minor chord.


The additive inverse of a minor chord is a major chord. Since it is similar to the proof above, it is left as an exercise to the reader. This proof can also be extended to any major chord in any voicing, since any major chord can be represented as v2 by adding integer multiples of 12 to each entry.


The proof for augmented and diminished chords is thankfully easier, becuase in both tonalities, intervals between adjacent notes are the same. A diminished seventh chord can be represented with respect to a root r1 as v1 = [r1, r1 + 3, r1 + 6, r1 + 9]. Taking the negative of this vector results in [-r1, -r1 - 3, -r1 - 6, -r1 - 9]. Add 9 to all entries, resulting in [-r1 + 9, -r1 + 6, -r1 + 3, -r1]. Set r1 = -r2, resulting in [r2 + 9, r2 + 6, r2 + 3, r2]. The entries in the vector match those in v1, so the chord remains a diminished seventh chord. The proof for augmented chords is similar and is left as an exercise to the reader.
